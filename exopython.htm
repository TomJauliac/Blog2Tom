Décomposition

nombre = input("il faut donner un nombre :")
nombre = int(nombre)
def decompNombre (nombre : int, base : int ) -> str :
    chaineSymbole = ""
    while nombre!= 0 :
       reste = nombre % base
       nombre = nombre // base
       chaineSymbole = str(reste) + chaineSymbole
       print(" Nombre = ", nombre, " r = ", reste)

    return chaineSymbole

for base in range (2,11) :
    print(nombre, " en base ", base, " donne ", decompNombre(nombre, base))

  • Les deux premières lignes servent à introduire un nombre, la première nous demande de choisir un nombre et la deuxième introduit ce nombre dans l'algorithme.
  • La boucle "for" à la fin sert à demander aux calcul situé juste au-dessus (c'est à dire qu'on se sert de la variable nombre ainsi que de la fonction decompNombre) de convertir le nombre choisi dans les bases de 2 à 10.
