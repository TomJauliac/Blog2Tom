    Le # permet de laisser un commentaire dans Python.

    • Exemple : décomposition d'un nombre en ses symboles

    Etude du problème

    Décomposons le nombre 7843 Pour obtenir le chiffre (symbole) on réalise une opération de division 7843 = 784 * 10 + 3

    Pour les unités :
    7843 % 10 = 3 (3 est le reste)
    7843 // 10 = 784 (784 est le quotient)
    Pour les dizaines :
    784 % 10 = 4 (4 est le reste)
    784 // 10 = 78 (78 est le quotient)
    Pour les centaines :
    78 % 1 0= 8 (8 est le reste)
    78 // 10 = 7 (7 est le quotient)

    La boucle "For" (pour)

    liste = []
    for i in range (2,200,2) :
        liste += [i]
    print("liste = ", liste, "\n de longueur", len(liste))
